# 🎬 Movie Recommendation System

## Overview

The Movie Recommendation System is a Machine Learning project that recommends movies based on their similarity to a selected movie. The system uses Content-Based Filtering and Cosine Similarity to generate personalized movie recommendations.

## Features

* Movie recommendation based on genre similarity
* Interactive user interface built with Streamlit
* Fast recommendation generation using Machine Learning techniques
* Simple and user-friendly design

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## Installation

1. Clone the repository:

git clone https://github.com/hemand18/Movie-Recommendation-System.git

2. Navigate to the project directory:

cd Movie-Recommendation-System

3. Install required libraries:

pip install -r requirements.txt

## Running the Application

Start the Streamlit application:

streamlit run app.py

The application will open automatically in your browser.

## Machine Learning Approach

The project uses:

* Content-Based Filtering
* Count Vectorization
* Cosine Similarity

These techniques analyze movie genres and recommend movies with similar characteristics.
